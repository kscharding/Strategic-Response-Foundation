ARD Action Pack

The actions in the ARD Action Pack augment the set of actions shipped with Remote Desktop to include some common setup and media presentation options. Here's a description of the actions:

  1. Display Message - This action will display a message or computer information full screen on the chosen remote computers. Useful for visually identifying classroom computers. Requies the installation of the Screen Message application, included with the ARD Action Pack download, on the remote computers.
     
  2. Start Application - This action will start an application on the remote computers passed from the previous action. Targets a list of Apple applications.
     
  3. Force Quit Application - This action will force quit an application on the remote computers passed from the previous action. Targets a list of Apple applications.
     
  4. Set QuickTime Player Prefs - This action sets the preferences of the QuickTime Player application on the remote computers passed from the previous action.
     
  5. Close All Windows in QuickTime Player - This action will close all windows in the QuickTime Player on the remote computers passed from the previous action.
     
  6. Open URL in QuickTime Player - This action will display the provided URL in the QuickTime Player on the remote computers
     
  7. Copy to Movies Folder - This action will copy the chosen movie files to the Desktop folder on the computers passed from the previous action. The copied movie files will then be moved into the Movies folder on the remote computer. Any conflicting movie files on the target computer's desktop or Movies folder will be replaced with the copied files.
     
  8. Present Remote Movie - This action presents a movie on the remote computers passed from the previous action. The movie must either be the frontmost movie currently open in the QuickTIme Player on the remote machine, or be a movie file located in the Movies folder of the remote computer.
     
  9. Close Every Safari Window - This action will close every open Safari window on the passed remote computers.
     
  10. Open URL in Safari - This action will display the provided URL in Safari on the remote computers.
     
  11. Set Browser Dimensions - This action will set the dimensions of the frontmost Safari browser window on the computers passed from the previous action.
     
  12. Open System Preference - This action will open the chosen System Preference pane on the computers passed from the previous action.
     
  13. Observe Computers - This action will open a new observation window in Remote Desktop for the computers passed from the previous action. The observation window can be zoomed to full screen.
     
  14. Stop Message Display - This action will cancel the display of messages on remote computers.
  15. Set Finder Access Options - This action controls the display of standard Finder commands that provide access to preferences, directories, and servers.
  16. Set Dock Access Options - This action will enable the setting of Dock preferences that enable or disable access to special settings.
